# Nodul Whitelabel Example

Этот проект создан с использованием React.

## Установка зависимостей

Перед запуском убедитесь, что у вас установлен [Yarn](https://classic.yarnpkg.com/lang/en/docs/install/).

Установите зависимости с помощью команды:

```bash
yarn
````

## Запуск проекта

Для запуска проекта выполните:

```bash
yarn start
```

После запуска приложение будет доступно по адресу:

[http://localhost:3000](http://localhost:3000)
